# 블랙잭 

### 게임준비

1. 카드는 4개의 무늬(하트, 스페이드, 클로버, 다이아)가 A(1) ~ 11까지 존재한다. 
   1. King, Queen, Jack 도 존재하면 각각10으로 계산한다.
   2. Ace는 1또는 11로 계산이 가능하다.

2. 게임에 필요한 인원을 입력받는다.

3. 카드를 2장씩 나눠준다.
   1. 카드는 한정적이고 똑같은문양 똑같은 번호의 카드는 줄 수 없다.
      - 만약 똑같은 카드를 드로우 할 시 null 값이 출력되며 다시 드로우를 해야한다.
